Competitor review intelligence for the MD/DC flooring & carpeting market — mirror the market's language and discontent.

== Description ==

Pulls Google reviews for the competitor target list via DataForSEO (Business
Data API) and mines them with zero external AI services:

* **Language Bank** — the exact phrases behind 5-star praise and 1-star anger
* **Discontent Map** — negative mentions per competitor across 8 themes
  (scheduling, communication, pricing, workmanship, cleanliness, crew,
  warranty/follow-up, sales)
* **Opportunities** — competitor pain ranked by volume, each with target
  keywords and a ready-made page angle

= Connections =

* **Midland Local SEO** — reuses its DataForSEO credentials (no second key)
  and lives under its admin menu
* **Midland Smart SEO** — one click sends opportunity keywords into the
  keyword cluster tool, and one click generates the draft page through the
  programmatic page generator (Elementor template + schema included)

= Requirements =

The DataForSEO plan must include the Business Data API (Google Reviews) —
it is a separate product from the SERP API used by the geo-grid. If the plan
lacks it, the fetch button explains exactly that.
