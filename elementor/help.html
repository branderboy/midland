Midland Floor Care — Sales Page Kit

1. Import the templates

Full kit (Elementor Pro): WordPress admin → Elementor → Tools → Import / Export Kit → Import Template Kit → upload midland-elementor-kit.zip. Brings home page + Theme Builder header + Theme Builder footer.

Single page only: Elementor → Templates → Saved Templates → Import → upload templates/home.json.

2. Paste the stylesheet

This kit's native Elementor widgets handle the structure; midland-styles.css handles the typography, gradient overlays, photo frames, and hover animations that aren't built into native widget controls.

Elementor Pro: Elementor → Site Settings → Custom CSS → paste contents of midland-styles.css.

Free Elementor: Appearance → Customize → Additional CSS → paste it there instead.

3. Hook up the form

The "Schedule a visit" section uses a shortcode placeholder: [smart-forms-pro id='1']. Replace the form ID with your actual Smart Forms Pro form ID, or swap the shortcode for your form plugin of choice (Contact Form 7, WPForms, Elementor Pro Forms).

4. Images

All template images load from raw.githubusercontent.com/branderboy/midland/main/images/. They render immediately on import. To host them on your own site, upload the files to your media library and replace the URLs.
