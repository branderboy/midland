Midland Floor Care — Sales Page Kit

Single-page Elementor template kit for the Midland Floor Care sales page.

Importing

  1. Zip the contents of this folder (manifest.json + templates/ + screenshots/) into midland-elementor-kit.zip.
  2. WordPress admin → Elementor → Tools → Import / Export Kit → Import Template Kit.
  3. Upload the zip and finish the import wizard.

Single page only?

If you just want the home page JSON, import templates/home.json directly via Elementor → My Templates → Import.
